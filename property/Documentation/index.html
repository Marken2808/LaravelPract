Installation information

After buying Real Estate Custom Script, Follow these steps for installing Script For your website or Portal

Step 1 → Downloading Realestate script Files From codecanyon.net

1 Extract file “Realestate_Custom_script_package.zip” using Extract software (winrar, 7zip or other software) you will get these folder - RealEstate_custom_script.zip, documentation, screenshot or Mysql Files.

Installation via FTP.

You can use software such as Filezilla for this. Upload the extracted Real Estate folder into public_html or www according to your server.
Make sure. All files must upload. if any file(s) not uploaded than script may not work properly.

Step 2 → Also make a database on your server Database wizard. We need database to install Real Estate Custom Script. This database will use to store all data of your website.

Step 3 → Open Install Location in your Web Browser.
